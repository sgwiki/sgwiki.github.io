# 작성 지시

1. reference 자료에서 **공식 인용**(게임 내 대사, TIPS, 공식 인터뷰, 공식 QA)을 식별하여 `[공식]` 태그로 분류한다.
2. reference 자료에서 **팬 해설** 부분을 식별하여 `[팬 분석]` 태그로 분류한다.
3. `wiki/_template/main.md` 템플릿에 따라 메인 문서를 작성한다.
4. 클릭 뎁스가 필요한 경우 `wiki/_template/depth.md` 템플릿에 따라 하위 문서도 작성한다.
5. 모든 인용에 원문 출처와 링크를 포함한다.
6. 개요는 비전공자도 이해할 수 있는 3~5문장으로 작성한다.
7. 문서 말미에 검수 체크리스트를 포함한다.
