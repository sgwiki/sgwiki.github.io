## 핵심 용어 (Convergence 관련)

| 항목 | 올바른 표기 | 주의 표기 | 비고 |
|------|------------|-----------|------|
| 수속 (convergence) | **수속** | 수순, convergence (영어 병기) | 작품 표기: '수속 사건', '수속된다' |
| 수속 결과 | **수속 결과** | 수순 결과, 수순 | convergence의 번역 표기 |
| 어트랙터 필드 | **어트랙터 필드** | attractor field (혼용) | 한글 표기 우선 |
| 수속 미래 | **수속 미래** | 수순 미래, 향하는 미래 | '수속' 표기 사용 |

**주의:** '수순'으로 변경하면 의미가 달라질 수 있음('절차'와 혼동). 작품 공식 표기인 '수속'을 유지할 것.

## 주의해야 할 오타 / 어색한 표현

| 원문(오타) | 올바른 표현 | 비고 |
|-----------|-----------|------|
| 폰웨이브(이름 가변) | **전화렌지(가칭)** | 가제(Gag) 표기: 오카베가 붙인 가칭 명칭 |
| 운명석의 문슈타인즈 게이트 | **운명석의 문(슈타인즈 게이트)** | 뭉쳐진 오타, 분리 필요 |
| 해결구 | **결말, 결말**, 해결 | '해결구'는 모호한 표현 |
| 결사의 시간 | **결사적 시간** | '결사'는 명사, '결사적'이 관형사로 자연스러움 |
| 보여준다 | **펼친다, 드러낸다** | 어휘 다양성 고려 |
| 집이 근처라 | **집이 가까워** | 자연스러운 한국어 표현 |
| 수속에 대한 두려움 | **수속에 대한 두려움** | (영어 병기 불필요, 작품 표기에 맞춤) |

## 모호한 표현 → 명확한 표현

| 모호한 표현 | 명확한 표현 |
|-----------|-----------|
| 해결구 | 결말, 결말, 해결 |
| 구하려 하지만 | 구하려 했지만 |
| 여기서 (맥락 불명) | 이 과정에서, 이 시점에서 |
| 보여준다 (단순 나열) | 펼친다, 드러낸다, 보여주고 있다 |
| 돌아가도 | 되돌아가도 (타임리프 맥락) |
